# poly-charts
Poly relayer charts

### Get Started 

```
NAME:
   charts - Generate poly charts

USAGE:
   server [global options] command [command options] [arguments...]

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --file value  Data input (default: "data.txt")
   --path value  Data output (default: "data")
   --help, -h    show help (default: false)
   
```
